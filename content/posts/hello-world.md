I have been trying to maintain a personal blog for a while now. I short list domain names, write draft posts and keep making design changes for the website. But I never get around actually publishing anything in the end.

Today I have decided to end this. I won't make any more designs or choose another domain name. I will stick to the current one and I will post whatever I write no matter how bad. Perfection is probably only meant to be pursued, but never achieved.

A few good things happened doing this though. I learned CSS a little better. Without this, I wouldn't have learnt about flex box model or grid layout. I also tinkered with a lot of static site generators. These are fun tools. In the end I settled for [Hugo](https://gohugo.io/) for it's speed.

Alright that's all for now. I will publish another post soon.
